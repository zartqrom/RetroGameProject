# RetroGameProject ⚔

RetroGameProject is a Python project to emulate a retro console using mednafen as a core.

## Installation

Download the project, unzip it and create the folder <code>/home/pi/Docmuents/Games</code> that contains all your ROMs.

## Usage

Open a terminal and type <code>sudo <path_where_is_the_projetc>/dist/retroGame</code>
